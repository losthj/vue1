Vue的介绍 vue2.X

vue简介
vuex是一个专门为vue.js应用程序开的状态管理模式
它采用集中式存储管理应用的所有组件的状态

vue安装工程化工具：
npm install -g @vue/cli
# OR
yarn global add @vue/cli

vue创建一个项目：
vue create my-project
# OR
vue ui
